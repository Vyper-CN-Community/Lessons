# 欢迎使用 Remix IDE：你的 Web3 开发环境

欢迎！本课将介绍 Remix IDE（集成开发环境），这是一个强大的基于浏览器的工具，将成为我们接下来工作的核心。Remix 之所以特别有效，是因为它能帮助你清晰地可视化智能合约及其交互。

在整个课程中，尤其是在深入研究 Vyper 语法时，你将与 Smart Contract Programmer（在 YouTube 上关注他：@smartcontractprogrammer）一起学习，他也是 Cyfrin 团队的一员。你所获得的 Vyper 知识将直接应用于 Remix，用于构建我们的智能合约。

## Vyper 开发的初始 Remix IDE 设置

当你第一次启动 Remix IDE 时，你可能会看到一个 "Welcome to Remix IDE" 的提示，询问你的主要用例。对于本课程，请选择 **"Learning - discovering web3 development"**（学习 - 探索 web3 开发）。

你可能还会遇到一个 Remix IDE 教程弹出窗口。这个互动指南提供了有用的提示，并介绍了各种界面元素，如 Solidity 编译器和部署器选项卡。随意点击浏览以快速了解概况。

在左上角，你会找到 **文件资源管理器 (File Explorer)** 图标（类似于堆叠的文件）。这是你管理项目文件和文件夹的地方。默认情况下，Remix 会创建一个工作区 (`default_workspace`)，其中填充了主要与 Solidity 和 JavaScript 开发相关的示例文件和文件夹：

*   `contracts/`: 包含示例 Solidity (`.sol`) 文件。
*   `scripts/`: 包含示例部署脚本 (`.ts`)。
*   `tests/`: 包含示例测试文件 (`.sol`, `.js`)。
*   `.prettierrc.json`: 代码格式化配置文件。
*   `README.txt`: 一个文本文件。

然而，本课程专注于 **Vyper**，一种 Python 风格的智能合约语言。我们的目标是利用类 Python 的语法，所以我们不会使用默认的 Solidity 示例。

**建议**：为了避免混淆并为 Vyper 开发重新开始，最好清理默认工作区。你可以通过右键单击文件或文件夹并选择 "Delete"（删除），或者选中项目并点击文件资源管理器工具栏中的垃圾桶图标（"Delete item"）来删除这些默认项目。请继续删除默认的 `contracts`、`scripts`、`tests` 文件夹，以及 `.prettierrc.json` 和 `README.txt` 文件。

## 在 Remix 中使用文件和文件夹

清理完工作区后，让我们创建第一个 Vyper 文件。

1.  确保在文件资源管理器中选中了你的 `default_workspace`。
2.  点击文件资源管理器工具栏中的 "Create new file"（创建新文件）图标（看起来像带加号的页面）。
3.  将文件命名为 `favorites.vy`。

`.vy` 文件扩展名至关重要——它是文件后缀。这个扩展名告诉 Remix 和其他工具，该文件包含 Vyper 源代码，就像 `.sol` 表示 Solidity 或 `.py` 表示 Python 一样。

点击 `favorites.vy` 将在右侧的主编辑器面板中打开它，准备让你编写代码。

在文件资源管理器图标下方，有一个 "Search in files"（在文件中搜索）图标（放大镜）。这允许你在项目文件中进行搜索。例如，如果在 `favorites.vy` 中输入了 "hello"，你可以使用搜索功能找到它。你可以使用像 `*.sol`, `*.js`, `*.vy`（其中 `*` 作为通配符）这样的模式来指定包含在搜索中的文件类型。搜索结果将显示包含搜索词的相关文件和行号。

## 利用 RemixAI：你的 AI 助手（附带注意事项）

Remix 包含一个内置的 AI 助手 **RemixAI**，通常在右侧面板可见。像这样的 AI 工具对于学习、调试和加速开发非常有帮助。你应该习惯使用 AI 助手。

**然而，关键的一点是**：虽然 AI 非常擅长回答问题和提供建议，但你必须自己拥有扎实的基础知识。AI 模型可能会出错，尤其是在复杂或利基主题（如特定的智能合约语言特性）上。你的基础知识对于核查 AI 的输出并在其犯错时进行纠正至关重要。

## 激活 Vyper 编译器插件

默认情况下，Remix 主要为 Solidity 设置。要使用 Vyper，你需要激活 Vyper 编译器插件。

1.  点击左侧边栏中的 **插件管理器 (Plugin Manager)** 图标（通常看起来像一个电源插头）。
2.  在插件管理器的搜索栏中，输入 `vyper`。
3.  找到 "**VYPER COMPILER**" 模块（它可能会列在 "Inactive Modules" 下）。
4.  点击旁边的 "**Activate**"（激活）按钮。

激活后，一个代表 Vyper 编译器的新图标（通常类似于 Vyper 语言 Logo）将出现在你的左侧边栏中。此插件负责获取你的 Vyper 代码并将其翻译成以太坊虚拟机 (EVM) 可以理解的**字节码 (bytecode)**——这个过程称为**编译**。我们很快将更详细地探讨编译。

## 优化你的工作区视图

为了最大化你的编码区域并减少干扰，你可以折叠你当前未使用的面板。

*   双击左侧边栏中的图标（如新激活的 Vyper 编译器图标）通常会**切换**其对应面板的打开或关闭状态。
*   底部面板（通常显示终端或控制台输出）通常有一个 "Hide Terminal"（隐藏终端）按钮（通常是一个向下的箭头）来折叠它。

保持工作区整洁有助于你专注于 Vyper 代码。现在你的 Remix 环境已为 Vyper 设置完毕，你准备好开始学习语法并构建你的第一个智能合约了！
