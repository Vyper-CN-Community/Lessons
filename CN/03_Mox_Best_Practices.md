# 课程最佳实践：导航、资源与提问及其艺术

欢迎！为了确保你在 Vyper & Moccasin 课程中获得最顺畅的学习体验，本节课概述了访问资料、提问以及利用基本工具的最佳实践。遵循这些准则将帮助你保持条理，并有效地获得所需的支持。

## 访问课程资料与代码

所有课程资源都已集中管理，便于访问，主要通过 GitHub 进行。

### 1. 寻找主仓库 (Main Repository)

*   **在 Cyfrin Updraft 上 (updraft.cyfrin.io)**：找到右上角的 GitHub 图标下拉菜单，选择 "Course Resources"（课程资源）。
*   **在 YouTube 上**：查阅视频描述中的相关链接。
*   **直接链接**：本课程的主要枢纽是 GitHub 上的 `Cyfrin/moccasin-full-course-cu` 仓库：[https://github.com/Cyfrin/moccasin-full-course-cu](https://github.com/Cyfrin/moccasin-full-course-cu)

### 2. 浏览主仓库 (moccasin-full-course-cu)

*   该主仓库中的 `README.md` 文件是你的核心指南。
*   它包含一个链接到不同课程章节的**目录 (Table of Contents)**。使用此目录导航到你感兴趣的特定课程或主题。

### 3. 寻找特定章节的代码

*   主仓库 (`moccasin-full-course-cu`) 提供结构和链接，但它**不直接包含**每一课的最终代码。
*   在主 `README.md` 中，目录中列出的每个部分（例如 "Vyper 101" -> "Section 1: Welcome to Remix - Favorite's List"）都会有一个专门的 **"Code:"** 链接。
*   **重要提示**：此 **"Code:"** 链接指向一个**单独的、特定的 GitHub 仓库**，该仓库保存了该特定章节的完整代码。
*   **示例**：对于 "Section 1: Welcome to Remix - Favorite's List"， "Code:" 链接将你指向 `Cyfrin/remix-favorites-cu` 仓库 ([https://github.com/Cyfrin/remix-favorites-cu](https://github.com/Cyfrin/remix-favorites-cu))。在这个仓库里，你会找到该章节开发完成的最终代码，例如 `Favorites.vy` 文件 ([https://github.com/Cyfrin/remix-favorites-cu/blob/main/Favorites.vy](https://github.com/Cyfrin/remix-favorites-cu/blob/main/Favorites.vy))。请使用这些特定章节的仓库来查看课程各部分的最终参考代码。

## 获取帮助与有效提问

我们鼓励社区互动和提问，但请务必使用正确的渠道。

### 1. 在哪里提问

*   所有问题、讨论、澄清请求或建议都应发布在**主课程仓库**的 **Discussions (讨论)** 标签页中：[https://github.com/Cyfrin/moccasin-full-course-cu/discussions](https://github.com/Cyfrin/moccasin-full-course-cu/discussions)
*   这是与课程内容相关的社区互动和支持的中心。

### 2. 哪里 **不** 该提问

*   **不要**在特定章节的代码仓库（如 `remix-favorites-cu`）上创建 "Issues" 或 "Pull Requests" 来提问。这些仓库仅用于托管每一课的最终参考代码。请改用主仓库的 Discussions 标签页。

### 3. 如何提问

我们鼓励提出清晰、结构良好的问题，以便从社区和 AI 工具那里获得最有效的帮助。

*   请参考位于主课程仓库中的专用指南 `how-to-ask-a-question.md`：[https://github.com/Cyfrin/moccasin-full-course-cu/blob/main/how-to-ask-a-question.md](https://github.com/Cyfrin/moccasin-full-course-cu/blob/main/how-to-ask-a-question.md)
*   本指南提供了提出有效问题的技巧，无论你是与讲师、同学还是 AI 助手互动。

## 成功的必备工具

为了最大化你的学习效果并与行业实践接轨，我们强烈建议使用以下工具：

### 1. GitHub 账户

*   如果你还没有，强烈建议创建一个 GitHub 账户。
*   对于 Web3 中的任何技术角色，包括智能合约开发，熟练掌握 Git 和 GitHub 至关重要。

### 2. AI 助手 ("AI Frens")

*   本课程结合了 AI 工具的使用来辅助学习和开发。
*   推荐工具（你可以在主仓库 README 的 "Resources For This Course" -> "AI Frens" 下找到列表）包括：
    *   `claude.ai`
    *   `ChatGPT` (课程示例中经常使用)
*   熟悉这些工具可以显著提升你的开发工作流。
*   **注意**：由于通过 Cursor IDE 扩展出现的漏洞利用事件频发，我们目前**不推荐**使用 Cursor IDE。

通过遵循这些访问资源、提问和利用关键工具的最佳实践，你将为自己在 Vyper & Moccasin 课程中开启一段更高效、更具参与感的学习旅程做好准备。记住，所有问题请移步主仓库的讨论区 (Discussions)！
