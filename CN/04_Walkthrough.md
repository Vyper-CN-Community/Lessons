# 部署你的第一个 Vyper 智能合约

欢迎来到本课程的第一个实战编码环节！准备好，因为在本课结束时，你将实现一个重要的里程碑：使用 Vyper 编程语言编写并部署你的第一个智能合约。我们为你迈出进入区块链开发世界的这重要一步感到兴奋。

在本节中，我们将使用一个名为 `Favorites.vy` 的特定智能合约。你可以把它看作是一个简单的数字账本，用于存储一个幸运数字以及一个包含人名及其幸运数字的列表。我们将一步一步地构建这个合约。你可以在课程的 GitHub 仓库 (`github.com/Cyfrin/remix-favorites-cu`) 中看到这段代码的最终版本，包括注释、状态变量 (`my_favorite_number`)、结构体 (`Person`)、列表 (`list_of_people`)、映射 (`name_to_favorite_number`) 和函数 (`store`, `retrieve`, `add_person`)。

我们的主要工具将是 Remix 集成开发环境 (IDE)，可通过在线访问 `remix.ethereum.org` 使用。Remix 提供了一个便捷的平台，可以直接在浏览器中编写、编译和部署我们的 Vyper 代码。上述提到的 GitHub 仓库也包含有用的资源，包括 `README.md` 文件中直接在 Remix 中打开此项目的链接。

为了从本节中获得最大收益，**积极参与是关键**。我们强烈鼓励你按照说明**跟随编写代码**。不要只是阅读——要自己亲手敲代码。这种动手实践的方法对于深刻理解概念和真正弄懂这些系统如何工作至关重要。手动编写代码可以建立肌肉记忆和更深层次的理解，这是以后构建更复杂应用程序，甚至是有效指导或纠正 AI 生成代码所必需的基础。

记住这句格言：“重复是技能之母。” (Repetition is the mother of skill.) 在跟随的过程中，请随意暂停、重读章节并自行尝试。我们的目标不仅是让你看到代码，而是让你在结束本节时掌握技能。

让我们开始吧！通过跟随操作，你很快就会写出你的第一段 Vyper 代码，并使用 Remix 成功地将你的第一个智能合约部署到区块链环境中。
