# 欢迎来到实战演练：巩固你的 Vyper 技能

恭喜完成第一部分：欢迎来到 Remix - Favorite's List！为了帮助你巩固刚刚学到的概念，我们推出了一个名为“实战演练 (Workshop)”的经常性功能。

这些实战演练的主要目标很简单：**练习**。通过解决有针对性的练习，你将积极应用前几课所涵盖的原则。

## 推荐的实战演练工作流程：练习、暂停和继续

为了从这些实战演练中获得最大收益：

1.  **首先独立尝试**：花专注的时间（例如 25 分钟）使用你所学的知识自己解决实战演练提示。
2.  **知道何时暂停**：如果计时器耗尽而你卡住了，停下来。不要让挫折感占据上风。
3.  **明智地寻求帮助**：休息之后，利用 AI 助手、讨论论坛或社区 Discord 等资源。

## 实战演练提示：Remix Favorites List (favorites.vy)

基于在 "Favorite's List" 部分开发的 `favorites.vy` 合约，处理以下三个提示。

### 提示 1：创建一个 add 函数

*   **任务**：在你的合约中定义一个名为 `add` 的新函数。
*   **功能**：此函数每次被调用时，应将存储在 `my_favorite_number` 状态变量中的值增加 1。
*   **背景**：你正在添加一种修改存储数字的新方法。

```vyper
# 提示结构
@external
def add():
    # 你的逻辑在这里，以递增 self.my_favorite_number
```

### 提示 2：修改起始值

*   **任务**：更改合约首次部署时分配给 `my_favorite_number` 状态变量的初始值。选择除 7 以外的任何值。
*   **验证**：部署并使用 `retrieve` 检查该值。
*   **背景**：这涉及编辑 `__init__` 构造函数。

### 提示 3：创建并使用新的结构体（挑战）

此提示涉及多个步骤：

1.  **定义新结构体**：定义一个具有你选择字段的新自定义数据类型。
2.  **创建新状态变量**：声明一个此新结构体类型的状态变量。
3.  **创建结构体填充函数**：编写一个接受输入并创建结构体实例的函数，将其存储在状态变量中。

**加分项**：创建一个 `@view` 函数来检索存储在你的新结构体中的数据。
